## GRIPPADO DOTFILES

### Using:
- Shellscript
- ZSH
- Oh-My-ZSH
- FZF

Compatible with __macOS__ and __Homebrew__ and __Linux__ with __APT package manager__

------

#### Usage:

```shell
git clone git@github.com:grippado/dotfiles.git ~/.dotfiles

cd ~/.dotfiles

chmod r+x install.sh

./install/sh
```
