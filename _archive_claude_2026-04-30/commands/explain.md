Explain how `$ARGUMENTS` works. Follow these steps:

1. Find and read the relevant code
2. Trace the main execution flow
3. Identify key dependencies and interactions
4. Explain in clear, structured sections:
   - **What it does**: High-level purpose
   - **How it works**: Step-by-step flow
   - **Key dependencies**: What it relies on
   - **Entry points**: How it gets called/triggered
5. Keep the explanation concise and practical
